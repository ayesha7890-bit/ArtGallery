# 🎨 Online Art Gallery & Auction Platform

An ASP.NET Core MVC web application for browsing, listing, and auctioning artwork online. Built as a full-stack academic project with a focus on clean architecture and real-world functionality.

---

## 🚀 Features

- 🖼️ Browse and view artwork listings
- 📤 Upload and manage artwork (with image support)
- 🔐 User authentication (Register / Login)
- 🏷️ Auction system for buying and selling art
- 👤 User profile management
- 🔒 Role-based access (Admin / User)
- 📊 Admin panel for managing listings

---

## 🛠️ Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| Backend    | ASP.NET Core MVC (.NET) |
| Frontend   | Razor Views, HTML, CSS  |
| Database   | SQL Server (SSMS)       |
| ORM        | Entity Framework Core   |
| Auth       | ASP.NET Identity        |

## ⚙️ How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/ayesha7890-bit/ArtGallery.git
   ```

2. Open in Visual Studio

3. Update connection string in `appsettings.json`:
   ```json
   "ConnectionStrings": {
     "DefaultConnection": "Server=YOUR_SERVER;Database=ArtGalleryDB;Trusted_Connection=True;"
   }
   ```

4. Run migrations:
   ```bash
   dotnet ef database update
   ```

5. Press **F5** to run
